🚀 전체 시스템 실행 방법
0. 인스톨
pip install pandas qdrant-client sentence-transformers langchain langchain-openai fastapi uvicorn pydantic streamlit requests

1️⃣ 첫 번째 터미널: 백엔드 서버 켜기 (AI 두뇌)
VS Code (또는 사용하는 에디터)에서 터미널을 엽니다.

현재 위치가 프로젝트 최상위 폴더(my_rag_project 또는 univ_project2)인지 확인합니다.

이전에 만들어둔 가상환경((.venv))이 잘 활성화되어 있는지 확인해 주세요! (안 되어 있다면 Activate.ps1 스크립트를 먼저 실행하셔야 합니다.)

아래 명령어를 복사해서 붙여넣고 엔터를 칩니다.

Bash
python -m uvicorn api.main:app --reload
터미널 창에 초록색으로 Application startup complete. 메시지가 뜨면 백엔드가 성공적으로 켜진 것입니다.

주의: 이 터미널 창은 절대 끄면 안 됩니다. 이대로 최소화하거나 가만히 둡니다.

2️⃣ 두 번째 터미널: 프론트엔드 서버 켜기 (UI 화면)
VS Code 터미널 창의 우측 상단에 있는 + 버튼 (또는 Split Terminal 버튼)을 눌러서 새로운 터미널 창을 하나 더 엽니다. (이제 터미널 창이 2개가 되었습니다!)

새로 연 터미널 창에서도 역시 가상환경((.venv))이 켜져 있는지 꼭 확인해 주세요.

방금 만들어둔 frontend/app.py 파일을 실행합니다.

Bash
python -m streamlit run frontend/app.py

3️⃣ 웹 브라우저에서 서비스 확인하기
3번 명령어를 치는 순간, 시스템 기본 웹 브라우저가 자동으로 새 탭을 열면서 http://localhost:8501 주소로 이동할 겁니다.
